# Random Song
Adds two <cg>random song buttons</c> to the <cg>editor</c> when choosing a song:
- **<cg>Random Library Song</c>**: picks a random song from <cg>Geometry Dash's music library</c>.
- **<cg>Random Stored Song</c>**: picks a random song from your <cg>already downloaded songs</c>.

*You may disable automatic searching after clicking either of these buttons in the mod's settings.*
